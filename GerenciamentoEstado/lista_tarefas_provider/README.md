## 2. Introdução

### 2.1

Descrever os requisitos do projeto "GTD", aplicação mobile desenvolvida em flutter com uso **Provider**

### 2.2 Escopo

O Sistema ermitirá:
- cadastrar tarefas;
- visualizar a lista de tarfas
- alterar os status da tarefa
- remover tarefas concluídas
- visualizar indicadores consolidados em um dashboard

Conceito do projetp:
- modelagem de dados
- separação entre interface e regras de negócio (MVC)
-  gerenciameno de estado com Provider
